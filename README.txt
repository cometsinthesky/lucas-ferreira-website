# Lucas Ferreira Science & Physics

Lucas Ferreira Science &amp; Physics é um site criado para apresentar imagens, vídeos, jogos educacionais, simulações, materiais didáticos, publicações e projetos autorais ligados à Ciência e à Física. O projeto funciona como um portfólio digital acadêmico e educacional, reunindo recursos visuais e pedagógicos em uma interface elegante.
